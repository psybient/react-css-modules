## Ant.Design

### Install

```yarn add antd```

### Usage

```import { DatePicker } from 'antd'```

### Config:

```.babelrc {  "plugins": [    ["import", { "libraryName": "antd", "libraryDirectory": "es", "style": "css" }]  ]} ```

### Using antd component

```import { DatePicker, message } from 'antd'```
```import 'antd/dist/antd.css'```
